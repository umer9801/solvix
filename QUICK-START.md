# 🚀 Quick Start Guide - SEO Setup

## ⚡ 3-Step Launch (15 minutes)

### Step 1: Update Domain (5 min)
Find and replace `solvixcore.com` with your actual domain in:
- `app/layout.tsx`
- `lib/seo-utils.ts`
- `app/sitemap.ts`
- `app/robots.ts`

### Step 2: Add Verification (5 min)
In `app/layout.tsx`, add your Google Search Console code:
```typescript
verification: {
  google: 'your-code-here',
}
```

### Step 3: Deploy & Submit (5 min)
1. Deploy your site
2. Go to [Google Search Console](https://search.google.com/search-console)
3. Add property & verify
4. Submit sitemap: `https://yourdomain.com/sitemap.xml`
